Description

Aims and Fit of Module(i.e. relationship to programme)
To introduce students to the components of computer systems. 

To introduce students to the various architecture levels of computer systems, including the digital logic level, the microarchitecture level, the instruction set architecture level, and the assembly language level. 

To introduce a common CPU architecture (e.g. AMD, ARM, or Intel) so that students can relate theoretical concepts with their realisation in state-of-the-art processors.
Learning Outcomes(for accreditation and other reasons,sub-headings could be added to this section)
Students completing the module successfully should be able to:
1.To understand the components of computer systems, their function, and interaction;

2.To understand the various architecture levels of computer levels of computer systems, their function, and interaction.
Method of Teaching and Learning
Students will be expected to attend between two and three hours of formal lectures as well as to participate in up to one hour of supervised practicals in a computer lab in a typical week. Lectures will introduce students to the academic content and practical skills which are the subject of the module, while computer practicals will allow students to practice those skills.In addition, students will be expected to devote roughly two hours of unsupervised time to solving continuous assessment tasks and private study per week. Private study will provide time for reflection and consideration of lecture material and background reading.Continuous assessment will be used to test how far practical skills have been learnt. A written examination at the end of the module will assess the academic achievement of students.
Syllabus
Lectures 1-3  Introduction                     

                                  Overview and history of computer architecture; Design Principles for Modern Computers; Fundamental Buildings Blocks.

 Lectures 4-9 Computer Systems Components: Processors                      

                                  Processors (CPU Organization, instruction execution, RISC versus CISC, instruction-level parallelism, processor-level parallelism); Example Processors (Pentium, UltraSPARC, picoJava); Motherboards (Design, Form, Form Factors, Components); Chipsets and Controllers (North Bridge, South Bridge, Controller Chips, Chipsets); BIOS and Boot Processor ;

Lectures 10-13 Computer Systems Components: Primary Memory                       

                                 Primary Memory (Bits, Memory Addresses, Byte Ordering, Error-Correcting Codes, Cache Memory, Memory Packaging and Types)

Lectures 14-16 Computer Systems Components: Secondary Memory and Peripherals. Secondary Memory (Memory Hierarchies, Disks, RAID, Optical devices: CD/DVD); Input/output (Buses, Output devices including Raster and vector graphics systems and video display devices, Input Devices, Modems, Character Codes); Expansion Cards; Video Cards. 

Lectures 17-24Computer Systems Architecture: The Digital Logic Level                          

Gates and Boolean Algebra (gates, boolean algebra, implementation of boolean functions, circuit equivalence, minimization, sum of product forms); Physical Considerations (gate delays, fan-in, fan-out). Basic Digital Logic Circuits (integrated circuits, combinational circuits, programmable logic arrays (PLAs), arithmetic circuits, clocks); Memory (latches, flip-flops, registers, memory organisation, memory chips, RAMs and ROMs); CPU Chips And Buses (CPU chips, computer buses, bus width, bus clocking, bus arbitration, bus operations); example Buses (PCI, USB); Interfacing (I/O chips; address decoding)

Lectures 25-29  Computer Systems Architecture: The Machine Program Levels                                                    Overview of the Microarchitecture, Instruction Set, and Assembly Language Level of Computer Systems; Overview of Data Types and their Representation; Instruction Formats; Addressing; Flow of Control; Introduction to assembly languages; Assembly Process.

Lectures 30-34  Memory system organisation and architecture                          

                           Storage systems and their technology; coding, data compression, and data integrity; memory hierarchy; main memory organisation and operations; latency, cycle time, bandwidth, and interleaving; cache memories (address mapping, block size, replacement and store policy); virtual memory (page table, TLB); fault handling and reliability.

 Lectures 35-38  Interfacing and communication                          

                            I/O fundamentals (handshaking, buffering, programmed I/O, interrupt-driven I/O); interrupt structures (vectored and prioritised, interrupt acknowledgement); external storage, physical organisation, and drives; buses (bus protocols, arbitration, direct-memory access); introduction to networks; multimedia support; RAID architectures. 

Lectures 39-42  Case studies                          

                            Compontents and Architecture of state-of-the-art computer systems.
